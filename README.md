**LeetCode problems, but make it for Program Analysis, Transformation and Code Generation (PATCG).**

I got bored of grinding LeetCode like everyone else.
I love PATCG, the beautiful, brutal world of compilers, JITs, GCs, linkers, and everything that actually makes code run.

So I did the only logical thing: took 100+ LeetCode problems and mapped each one to its natural systems counterpart.

### What you'll find here
Every classic LeetCode problem re-imagined as a real systems programming challenge:
- **Two Sum** -> Symbol table resolution / constant pooling in the linker
- **Trapping Rain Water** -> Heap fragmentation & GC compaction
- **Merge K Sorted Lists** -> VTable / MRO resolution
- **Course Schedule** -> Circular dependency detection in module loading
- ...and 96 more

Made with pure boredom + love for PATCG.
